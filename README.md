# Email Spam Detection System
This project uses **Natural Language Processing (NLP)** and **Machine Learning** to classify emails as either 'Spam' or 'Ham'.

## Project Pipeline
1. **Scoping**: Problem description and goals.
2. **Data**: Loading and cleaning using NLTK and Scikit-Learn.
3. **Model**: Training a Multinomial Naive Bayes classifier.

## How to Run
1. Install dependencies: `pip install pandas scikit-learn nltk seaborn`
2. Open `main.ipynb` in VS Code.
3. Run all cells to train the model and see evaluation metrics.

## Performance
- **Accuracy**: ~99.2%.
- **Model**: Naive Bayes.
